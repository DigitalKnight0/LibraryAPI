# Nodejs Express base API 

** This is express base API buid with express**

## Getting Started

### Installation

Clone the repo:

```bash
git+ssh://git@gitlab.com/trungnq89/nodejs-api.git
cd nodejs-api
```

Install the dependencies:

```bash
yarn install
```

### Commands

Running locally:

```bash
yarn dev
```

Running in production:

```bash
yarn start
```

### Documentation

Swagger documentation generates at: http://localhost:<config.port>/v1/docs
